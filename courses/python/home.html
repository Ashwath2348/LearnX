Welcome to Python!

Did you know? Python is one of the most popular programming languages in the world!

What is Python?

Python is a high-level, interpreted programming language known for its simplicity and readability.

Common Uses:

  • Web Development (Django, Flask)
  • Data Science (Pandas, NumPy)
  • Automation & Scripting
  • AI & Machine Learning (TensorFlow, PyTorch)

Getting Started

Installing Python

  1. Download Python from python.org
  2. Run the installer
  3. Verify installation in your terminal:
python --version

Your First Python Program

The traditional "Hello World" program:

print("Hello, World!")

The print() function displays output to the console.

Variables and Data Types

Python has several basic data types:

# String (text) name = "Alice" # Integer (whole number) age = 25 # Float (decimal number) price = 9.99 # Boolean (True/False) is_student = True
